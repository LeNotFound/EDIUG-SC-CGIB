# ACM实验室

## 所属
信息工程学院

## 主要工作
- 选拔、训练和培养程序设计竞赛选手
- 组织算法竞赛培训与技术交流
- 参与和承办校内外各类程序设计竞赛
- 科普编程与算法相关知识

## 团队简介
北京印刷学院 ACM-ICPC 集训队是从属于北京印刷学院信息工程学院的创新团队，代表学校参加国际大学生程序设计竞赛（ICPC）、中国大学生程序设计竞赛（CCPC）、蓝桥杯、团体程序设计天梯赛等赛事，并围绕于此组织选手的选拔、训练、培养以及科普活动。
